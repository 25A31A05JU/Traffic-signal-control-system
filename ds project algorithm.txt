Step 1:START

Step 2:DEFINE lanes:
       - Lane1, Lane2, Lane3 

Step 3:INITIALIZE:
       - All lanes as empty
       - cycle = 0
       - maxCycles 

Step 4: ADD VEHICLES
        Insert vehicles into each lane
        
Step 5: DISPLAY LANES
        1. For each lane:
        a. Print all vehicles in order

Step 6:CHECK EMERGENCY (AMBULANCE)
        1. For each lane:
        a. Traverse the lane
        b. If "Ambulance" is found:
             i. Remove it from its position
            ii. Insert it at the front of the same lane
           iii. Mark this lane as priority
            iv. Stop checking further lanes

Step 7: SELECT LANE FOR GREEN SIGNAL
       1. If any lane has ambulance:
         a. Select that lane
       2. Else:
         a. Compare sizes of all lanes
         b. Select lane with maximum vehicles

Step 8: MOVE VEHICLE (SIGNAL GREEN)
        1. From selected lane:
        a. Remove front vehicle (pop/dequeue)
        b. Print which vehicle passed

Step 9: REPEAT PROCESS
       1. Increment cycle
       2. Repeat Steps 5–8 until:
         a. cycle == maxCycles OR
         b. all lanes are empty

Step 10:STOP.
