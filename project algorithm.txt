Step 1: Start.

Step 2: Initialize multiple lanes  
       
Step 3: Insert vehicles into each lane.
       
Step 4: Repeat the following steps for each signal cycle:

      a. Display the current vehicles in all lanes.

      b. Check for the presence of an emergency vehicle (Ambulance) 
         in each lane.

     c. If an ambulance is found:
        i. Move the ambulance to the front of that lane.
       ii. Assign green signal to that lane.

     d. Else 
        i. Compare the number of vehicles in all lanes.
       ii. Select the lane with the maximum number of vehicles.
      iii. Assign green signal to that lane.

     e. Remove the front vehicle from the selected lane.
      
Step 5: Repeat until all lanes become empty.

Step 6: Stop.
